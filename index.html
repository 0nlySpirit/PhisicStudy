Carga Eletrica de Processos de Eletrização

Eletrostatica:

É o ramo da fisica que estuda as cargas eletricas e as suas ações

Carga Eletrica

É associada aos elementos que constituem o atomo

Os protons e os neutrons, localizam-se no núcleo do atomo, enquanto os eletrons se movimentam ao redor do núcleo do atomo, Numa região chamada Eletrosfera;

Protons Positivo ( + )
Neutrons Negativo ( - )

Corpo Neutro = ( Protons = Eletrons )
Corpo Positivo = ( Protons > Eletrons )
Corpo Negativo = ( Protons < Eletrons )

Principios da Quantização da Carga

Eletrica

A Carga eletrica é quantizada, ou seja é um multiplo inteiro de carga elementar ( nao pode ser um valor flutuante, decimal )

Acima esta representando a "Formula Quantica"

e = Carga Elementar
n = N° de Eletrons
+ = Perde Eletrons
- = Ganha Eletrons
Q = Quantidade de Carga Eletrica [C]
